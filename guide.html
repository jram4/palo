Palo Installation Guide

Follow these steps to set up Palo and start using it with your YouTube videos.

Prerequisite

Step 1: Access Google Cloud Console

Click the button below to navigate to the Google Cloud Console.

Go to Google Cloud Console

Step 2: Create API key

Click the center button to Create a new API key.

Step 3: Create API key in a new project

Press the button to create a API key in a new project.

Step 4: Copy the key to your clipboard

Press the button to copy the key to your clipboard.

Step 5: Open Palo settings

In the top bar of your chrome browser press the puzzle icon to show your extension. Then click on the Palo name to open the extension's menu.

Step 6: Paste and Save the API key

Click on the input box and paste the text in your clipboard. On Mac: press Command + v, or Windows: Ctrl + v. Then press save to store the API key.
